# Projeto-sds3

# ![DevSuperior logo](https://raw.githubusercontent.com/devsuperior/bds-assets/main/ds/devsuperior-logo-small.png) Semana Spring React - SDS 3.0
Realizada entre 03 a 09 de abril de 2021. Projeto desenvolvido sob a tutoria de [Nelio Alves](https://github.com/acenelio).
Aplicação Web Dashboard de vendas com gráficos [Apexcharts](https://apexcharts.com/docs/react-charts/) e exibição de tabela de vendas com paginação.
Backend em Java com Spring Boot, Frontend em ReactJS com Typescript e Banco de Dados Heroku PostgreSQL.
